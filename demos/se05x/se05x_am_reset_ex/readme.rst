=======================================================================
 SE05X Access Manager Reset example
=======================================================================

This example is used to demonstrate sending the reset command from
client applications to access manager module using 'SM_AmResetI2C' API.


Building the Demo
=======================================================================

- Build Plug & Trust middleware stack. (Refer :ref:`building`)
- Project: ``se05x_am_reset_ex``


Running the Example
=======================================================================

On Raspberry-Pi or iMX board, run as::

    ./se05x_am_reset_ex
