10.13.4.1.17. Struct multicast_package_t¶

  • Defined in File sems_lite_api.h

10.13.4.1.17.1. Struct Documentation¶

struct multicast_package_t¶

Information about package

Public Members

size_t multicastCommandsLen¶

Length of multicast commands (as pointed by pMulticastCommands). If multicastCommandsLen mismatches to contents of pMulticastCommands, there will be undesired behavior of system.

size_t multicastPackageNameLen¶

Length of multicast Package name

uint8_t multicastPackageVersion[2]¶

Version information of this MulticastPackage, describing the sum version over all contained content. In case exactly one SubComponent is contained this version is equal to the version of this SubComponent. Version information major.minor, both values in range 0 to 127. bypte 0 is Major value, byte 1 is Minor value.

size_t pkSemsCaAutLen¶

Length of pPkSemsCaAut

const uint8_t *pMulticastCommands¶

The complete Multicast Applet Loader Script (certificate, signature, encrypted and signed commands) in protobuf format.

const char *pMulticastPackageName¶

Giving a descriptive name to the content of the Multicast Package. It is possible that one Multicast Package contains multiple Applet binaries and also that a Multicast Package contains no Applet binary at all (e.g. for deletion).

const uint8_t *pPkSemsCaAut¶

The PkSemsCaAut key found in the SEMS_SP_AUT.txt file as PK.AP.AUT.

const uint8_t *pSignatureOverCommands¶

The signature over the multicast commands in an machine readable form. So it does not have to be parsed form the script commands. String encoding is chosen here, as mayna json parsers can not handle such large integer values.

const sub_component_metaData_t *pSubComponentMetaData¶

A list of subcomponents of this MulticastPackage, designating all Executable Load Files (ELFs) Contained. It usually contains one entry, but can have multiple in the case multiple dependent ELFs get modified. This list can be empty, e.g. for a KeyRotation or content deletion MulticastPackage.

uint32_t requiredFreeBytesNonVolatileMemory¶

Minimum required free Non Volatile memory in bytes that have to be free on the target device before execution of MulticastPackage.

uint32_t requiredFreeBytesTransientMemory¶

Minimum required free transient (RAM) memory in bytes that have to be present on the target device before execution of MulticastPackage.

uint32_t semsLiteAPIVersion¶

Version information of SEMS Lite Agent API Defined follow SEMS_LITE_AGENT_VER_MAJOR

size_t signatureOverCommandsLen¶

Length of signatureOverCommands

uint8_t target12Nc[6]¶

Target 12nc is a 12 digit numerical code identifying the target device where this MulticastPackage is intended to be executed on, as known to customers and used on EdgeLock2Go to identify device types.

uint8_t targetEntityID[16]¶

Entity ID, 16bytes long Binary Coded Decimal, of the target device where this MulticastPackage is intended to be executed on. It is an identifier of the key-set of the Multicast Applet Loader. As the key-set and related Entity ID can change over the lifetime of the device it is separated from the Commercial Name. This is stored to have it formatted in the BCD encoded and therefore recognizable form.
