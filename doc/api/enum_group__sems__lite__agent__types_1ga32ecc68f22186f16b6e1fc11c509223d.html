  • Defined in File sems_lite_agent_common.h

10.13.4.2.3.1. Enum Documentation¶

enum _sems_lite_status¶

Status of SEMS Lite update operation

Values:

enumerator kStatus_SEMS_Lite_Success¶

Operation was successful

enumerator kStatus_SEMS_Lite_ERR_COM¶

Communication Error

enumerator kStatus_SEMS_Lite_ERR_DoReRun¶

Update not completed please provide update package again.

enumerator kStatus_SEMS_Lite_ERR_NotApplicable¶

Update not applicable on this Chip/type.

enumerator kStatus_SEMS_Lite_ERR_DoRecovery¶

Update can not be completed. Please provide recovery package, to roll back to last working version.

enumerator kStatus_SEMS_Lite_ERR_Fatal¶

Unresolvable error.

enumerator kStatus_SEMS_Lite_ERR_NotEnoughNVMemory¶

Not enough NV memory.

enumerator kStatus_SEMS_Lite_ERR_NotEnoughTransientMemory¶

Not enough transient memory.

enumerator kStatus_SEMS_Lite_ERR_MinPreviousVersion¶

Current SE version not meet min previous version request.

enumerator kStatus_SEMS_Lite_ERR_OlderVersion¶

Older than current SE version

enumerator kStatus_SEMS_Lite_ERR_General¶

General error.
